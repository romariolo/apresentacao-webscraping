Qual o nosso objetivo?

O projeto visa desenvolver uma solução automatizada para a coleta, tratamento e armazenamento de dados públicos da Revista da Propriedade Industrial (RPI), disponibilizada pelo INPI.

O objetivo final é criar uma base de dados estruturada e enriquecida, pronta para ser utilizada em análises exploratórias e pela aplicação de técnicas de Inteligência Artificial para identificar padrões e extrair insights valiosos sobre registros de marcas no Brasil.

Download e Extração dos Dados

Uma função dedicada para baixar os arquivos .zip do site, extrair o conteúdo XML e gerenciar os arquivos.


def download_and_extract_zip(url, folder='downloads_xml'):
    zip_filename = os.path.basename(urlparse(url).path)
    xml_path = os.path.join(folder, f"{zip_filename.replace('.zip', '')}.xml")

    if os.path.exists(xml_path):
        print(f"Arquivo {xml_path} já existe.")
        return False

    try:
        response = requests.get(url, stream=True)
        response.raise_for_status()

        os.makedirs(folder, exist_ok=True)
        zip_path = os.path.join(folder, zip_filename)

        with open(zip_path, 'wb') as f:
            f.write(response.content)

        with zipfile.ZipFile(zip_path, 'r') as zip_ref:
            zip_ref.extractall(folder)

        os.remove(zip_path)
        return True
    except Exception as e:
        print(f"Erro ao processar {url}: {e}")
        return False

Parsing do XML para JSON

Esta é a função principal que lê o arquivo XML complexo e o transforma em um dicionário Python limpo e estruturado.


def parse_xml_to_json(xml_file):
    tree = ET.parse(xml_file)
    root = tree.getroot()
    processos_json = []

    for processo in root.findall('processo'):
        protocolo_id = int(processo.get('numero'))
        titular = processo.findtext('titulares/titular/nome-razao-social', '')
        marca_nome = processo.findtext('marca/nome', '')
        # ... extração de outros campos ...

        processo_data = {
            "protocolo_id": protocolo_id,
            "titular": titular,
            "elemento_nominativo": marca_nome,
            # ... outros campos ...
        }
        processos_json.append(processo_data)
    
    return processos_json

Extração do PDF de Classificação

Um segundo script usa 'pdfplumber' para ler a tabela oficial do INPI e criar nosso dicionário de referência.


def extract_classes_from_pdf(pdf_path):
    produtos = []
    servicos = []
    with pdfplumber.open(pdf_path) as pdf:
        for page in pdf.pages:
            tables = page.extract_tables()
            for table in tables:
                for row in table:
                    # ... validação e limpeza de dados da linha ...
                    code_num = int(row[0])
                    descricao = row[1]
                    if 1 <= code_num <= 34:
                        produtos.append({"codigo": code_num, "descricao": descricao})
                    elif code_num >= 35:
                        servicos.append({"codigo": code_num, "descricao": descricao})
    return {"produtos": produtos, "servicos": servicos}
